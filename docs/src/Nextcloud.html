Installing snap on Manjaro Linux

命令行安装snapd：

$ sudo pacman -S snapd

安装后，需要启用用于管理主快照通信套接字的systemd单元：

$ sudo systemctl enable --now snapd.socket

注销并再次登录，或者重新启动系统，以确保正确更新快照的路径。
